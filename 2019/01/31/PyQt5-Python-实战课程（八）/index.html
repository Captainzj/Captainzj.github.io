多窗口交互（2）：使用信号和槽

Presentation



  • 要点说明
    如果一个窗口A与另一个窗口B交互，那么A尽量不要直接访问B窗口中的控件，
    应该访问B窗口中的信号，并指定与信号绑定的槽函数

    例：如果A直接访问B窗口的控件，一旦B窗口控件发生改变，那么A和B的代码都需要变化

    如果A访问的是B中的信号，那么B中的控件发生了改变，只需要修改B中的代码即可
